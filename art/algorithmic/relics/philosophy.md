# Forged Totems

Forged Totems treats relics as compressed force-diagrams: tiny emblems that imply ritual engineering, not decoration. Every icon should feel like a machine rune excavated from a dead order.

The algorithm must rely on deterministic seed vectors and constrained harmonic distortion so small canvases remain legible. This is a meticulously crafted algorithm problem: silhouette first, spectral contrast second, ornament last.

Relic generation should use high-value edge glow against deep matte cores so icons survive small UI scales. The work must read as the product of deep computational expertise and painstaking optimization, not generic noisy circles.

At mastery level, each relic seed should appear handcrafted while remaining mathematically repeatable. The aesthetic target is precision-with-voltage: severe, luminous, and unmistakably intentional.
