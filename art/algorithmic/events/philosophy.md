# Rift Chronicles

Rift Chronicles frames events as catastrophic atmosphere snapshots, where hidden forces are implied through pressure, glow, and fracture. Each event plate should communicate narrative risk before text is read, turning a node choice into a visual commitment.

The process is deterministic and seeded by event identity. Layered trigonometric interference drives cloud banks and fault lines, while radial attenuation controls focal tension. This should look like a meticulously crafted algorithm, tuned by someone with deep computational expertise rather than random brushwork.

High contrast is mandatory: obsidian foundations, incandescent reds, ritual violets, and blinding mineral highlights. The algorithm should repeatedly prove mastery by balancing cinematic drama with clear title-safe zones and readable UI overlays.

The visual rhythm must feel inevitable, as if each frame were the product of painstaking optimization over many iterations. Emergence is allowed, chaos is allowed, but composition discipline should always signal a master-level implementation.
